QPay Client-ийн Документ
========================

qpay-client гэж юу вэ?
~~~~~~~~~~~~~~~~~~~~~~

**qpay-client** нь QPay төлбөрийн системтэй *Python* орчинд холбох процессыг илүү хялбар, найдвартай, уян хатан болгох зорилготой нээлттэй сан юм.  
QPay-ийн төлбөрийн системтэй интеграц хийхэд шаардлагатай өгөгдлийг шалгаж, токен авах болон шинэчлэх үйлдлийг автоматжуулж, ``sync`` болон ``async`` горимд ажиллах боломжийг бүрэн дэмждэг.  
Энэхүү сан нь хөгжүүлэгчдэд QPay-ийн REST API-г ашиглахад үүсдэг давтагдсан кодчилол, баталгаажуулалтын асуудлуудыг автоматаар шийдэж өгдөг.

Онцлогууд
~~~~~~~~~

- Асинхрон (`async`) болон синхрон (`sync`) клиент  
- Токенийг автоматаар шинэчилдэг  
- Өгөгдлийн бүтэц болон хариуг шалгах баталгаажуулалттай  
- Алдаа боловсруулах, лог бичилт, дахин оролдлогын системтэй  
- Python 3.9+ хувилбаруудад тохиромжтой

Эхлэх
-----

Эхний алхамууд
~~~~~~~~~~~~~~

- **Танилцуулга:** qpay-client-ийн зорилго болон үндсэн ойлголтууд  
- **Хурдан эхлэх (Quickstart):** `pip install qpay-client` командыг ашиглан суулгаж, богино хугацаанд QPay API-г холбох жишээ  
- **Лог тохиргоо:** Хэрэглээний үед гарах мэдээллийг бүртгэх логгинг тохируулах заавар

QPay-тэй ажиллах
~~~~~~~~~~~~~~~~

QPay платформ дээрээ интеграц хийхийн тулд QPay-н албан ёсны вэбсайтаас бүртгэл үүсгэж, API түлхүүр болон нууц үгийг авах шаардлагатай.  
Дэлгэрэнгүй зааврыг дараах холбоосоор үзнэ үү:

- [QPay платформд бүртгүүлэх заавар](https://qpay.mn)
